# Kids Studio — Starter

This is a minimal starter for the Simulation Studio MVP demo.

Tech: Vite + React + TypeScript + PixiJS + Blockly

Quick start (Windows / PowerShell):

```powershell
cd "d:/code/kids code/studio-starter"
npm install
npm run dev
```

What’s included:
- Minimal Vite React app
- `Canvas` component using PixiJS (example animation)
- `BlocklyPanel` placeholder with runtime injection
